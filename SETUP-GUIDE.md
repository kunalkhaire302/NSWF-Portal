# NSWF Wikipedia Portal Setup Guide

## Quick Start

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Open `http://localhost:3000`

## Tech Stack
- Next.js 15
- React 19
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Project Structure
- `src/app/page.tsx`: Main application entry
- `src/components/NSWFWikipediaPortal.tsx`: Core modular component handling navigation, Wikipedia draft rendering, and about sections.
- `src/app/globals.css`: Custom animations and Tailwind v4 theme variables.
